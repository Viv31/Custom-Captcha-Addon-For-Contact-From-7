== Description ==
A plugin for adding custom captcha for contact form7 with button disable functionality.
== How to Use ==

Just install and use

== Requirements ==
It can run on any wordpress version.

== Frequently Asked Questions ==

= How can I apply use captcha in my contact form 7 =
It is very lightweight captcha form just install plugin and it will display captcha code 
with captcha insert block.
